### Quick Questions


# What is the main difference between a list and a tuple?
Lists are mutable, tuples are immutable

# What is the convention for naming lists?
Use plural names

# What would be a good name for a list of cities?
cities

# How many elements can you store in a list?
As many as memory allows

# True or False? Elements in a list must all be of the same type (e.g. all strings or all integers)?
False

# What does Python's len function do? What types can it be used with?
Returns the number of items, used with lists, strings, tuples, etc.


### Lists Warmup

## Part 1

numbers = [10, 20, 40, 81]

# numbers[0]
10

# numbers[-1]
81

# numbers[3]
81

# 20 in numbers
True

# 1 in numbers
False

# numbers[1] + numbers[2]
60

# len(numbers)
4


## Part 2

# Change the first element of numbers to 1
numbers[0] = 1

# Change the last element of numbers to 4
numbers[-1] = 4

# Check if 1 is an element of numbers
1 in numbers
